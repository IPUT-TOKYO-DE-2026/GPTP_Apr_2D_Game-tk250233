# “Easy”shooting

## ゲームの内容（全画面前提のゲームです）
ハイスコアを目指す鬼畜シューティングゲーム
純粋に難易度が高いため、ゲーム後のスコアを使い自分自身を強化、アイテム購入ができる。
スキルがあり、それを用いりスコアを伸ばす。
各スキルにはそれぞれ制限があり上手く使いこなす必要がある。
敵の弾をそばでよけるとスコアが少し増える
生かさず殺しきらずがスコアを伸ばすコツ

## 操作説明
- WASDや矢印キーで移動
- スペースキーで弾を発射
-シフトキーで低速移動
-Bでボムを出し画面上の弾を破壊
-１で無敵ダッシュ（クールタイムあり）
-２短押しで敵を貫通できる貫通弾を発射
-２長押しで敵の弾を防ぐエリアを生成
-３で場面にある敵の弾を全て回復弾にし扇状に弾を発射
